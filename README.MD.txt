Intrucciones para compilar y ejecutar el proyecto

BackEnd. 

1. En el Package Explorer en eclipse o tu IDE de java favorito, ir hasta la ruta src/main/java 
3. click derecho en com.eduesqui  
4. run as > java aplication 


FrontEnd.
1. abrir terminal 
2. ir hasta el directorio frontend , cd frontend en windows. 
3. npm install 
4. ng serve 

 http://localhost:4200/ en explorador web.
